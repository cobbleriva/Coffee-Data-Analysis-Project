I created a data analysis project based on this dataset from Kaggle - https://www.kaggle.com/datasets/ahmedabbas757/coffee-sales. I used Claude AI to create me 12 business questions around this dataset. Using Bigquery, I first answered them and then created tables that displayed the results. I also made visualisations of these business questions in Tableau. All the relevant files can be found below.

Here were the business questions:

What is the total revenue generated across all store locations? 
Which product category (e.g. Coffee, Tea, Bakery) generates the most revenue? 
What are the top 5 best-selling products by number of units sold? 
Which month had the highest total sales revenue? 
What is the busiest hour of the day based on number of transactions? 
How does revenue differ between weekdays and weekends? 
Which store location has the highest total revenue? 
How many transactions did each store location process in total? 
What is the average transaction value (revenue per transaction) for each product category? 
Which product type has the highest average unit price? 
What is the total number of transactions per month, and is there a clear growth trend? 
Which store location sells the most units per transaction on average (i.e. highest average quantity)?
